1.3 详解

介绍几个常用的配置项：

title ：标题栏显示文字

width ：窗口宽度

height ：窗口高度

modal ：是否设置为模态窗体

模态窗体：打开此窗体后，不能对其他窗体进行操作

closable ：是否显示关闭按钮

maximizable ：是否显示最大化按钮

minimizable ：是否显示最小化按钮

html ：窗口显示的内容

resizable ：用户是否可以调整窗口的大小

bodyStyle ：窗口内容与边框的间距

buttonAlign ：窗口中 button 的对齐方式（left/center/right）默认是 right

介绍几个方法：

show ：窗口显示

hide ：窗口隐藏

close ：窗口关闭
