## 文字版

### **C 语言**

```txt
基本类型
    整型
        char 型
        short int
        int
        long int
        long long int
    浮点型
        float 单精度浮点型
        double 双精度浮点型
        long double 精度更高的浮点型
构造类型
    数组
    结构体 struct
    共用体 union
    枚举类型 enum
指针类型
空类型 void
```

### **Python**

```txt
Number 数字
    int 有符号整型
    long 长整型，也可以代表八进制和十六进制
    float 浮点型
    complex 复数
布尔类型
    True
    False
String 字符串
List 列表
Tuple 元组
Dictionary 字典
Set 集合（扩展）
```

###  **Java**

```txt
基本类型 Primitive Type
    Boolean 类型
    数值类型
        整数类型
            byte
            short
            int
            long
            char
        浮点类型
            float
            double
引用类型 Reference Type
    类
    接口
    数组
    null 类型
```

###  **JavaScript**

```txt
主要类型
    string
    number
    boolean
    null

    String
    Number
    Boolean
    Object
    Function
    Array
    Date
    RegExp
    Error
```
