1.u

作为程序员，看到 u 字符，首先想到的便是 Unicode 这个单词。没错，它确实也代表此含义。目的是告知后面字符串采用 Unicode 编码格式，应用对象自然是容易出现问题的中文字符串。

有时我们存储完成后，再次取用发现格式不一致，或者显示乱码等问题，使用 u 便很好解决了。
